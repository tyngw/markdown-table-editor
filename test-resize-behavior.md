# リサイズ動作テスト

このテーブルを使って、カラムリサイズ時にソートが発火しないことを確認してください：

| 名前 | 年齢 | 職業 |
|------|------|------|
| 田中 | 25 | エンジニア |
| 佐藤 | 30 | デザイナー |
| 山田 | 28 | マネージャー |

## テスト手順

1. VSCodeでこのファイルを開く
2. `Ctrl+Shift+P` → "Markdown Table Editor: Edit Table" を実行
3. 「名前」列のヘッダー右端のリサイズハンドル（細い線）をドラッグして幅を変更
4. ソートが発火しないことを確認（データの順序が変わらないこと）
5. 「名前」列のヘッダー部分（リサイズハンドル以外）をクリック
6. ソートが正常に動作することを確認（データが名前順にソートされること）
