## Usage

1. Open this file in VSCode
2. Right-click on the table
3. Select "Open Table Editor"
4. Verify that `<br/>` tags are displayed as line breaks
5. Edit cells to add more line breaks
6. Verify that newlines are converted to `<br/>` tags when saved
