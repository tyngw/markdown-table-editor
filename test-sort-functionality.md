# ソート機能テスト

新しいビューオンリーソート機能をテストしてください：

| 商品名 | 価格 | カテゴリ | 在庫数 |
|--------|------|----------|--------|
| りんご | 150 | 果物 | 50 |
| バナナ | 200 | 果物 | 30 |
| にんじん | 80 | 野菜 | 25 |
| じゃがいも | 120 | 野菜 | 40 |
| 牛乳 | 180 | 乳製品 | 15 |

## テスト手順

### 1. ビューオンリーソートのテスト
1. VSCodeでこのファイルを開く
2. `Ctrl+Shift+P` → "Markdown Table Editor: Edit Table" を実行
3. 「商品名」列のヘッダーをクリックしてソート
4. 画面右上に「📊 Viewing sorted data」パネルが表示されることを確認
5. ステータスバーに「Sorted by A (ASC) - View Only」が表示されることを確認

### 2. 元の順序への復元テスト
1. 右上パネルの「📄 Restore Original」ボタンをクリック
2. 元の順序（りんご、バナナ、にんじん...）に戻ることを確認
3. 右上パネルが消えることを確認

### 3. ソートの保存テスト
1. 再度「商品名」列をクリックしてソート
2. 右上パネルの「💾 Save Sort to File」ボタンをクリック
3. 「Sort order saved to file」メッセージが表示されることを確認
4. 右上パネルが消えることを確認
5. エディタを閉じて再度開き、ソートが保存されていることを確認

### 4. 数値ソートのテスト
1. 「価格」列をクリックしてソート
2. 数値順（80, 120, 150, 180, 200）にソートされることを確認
3. 再度クリックして降順（200, 180, 150, 120, 80）になることを確認

### 5. セル編集時の動作テスト
1. 「価格」列でソート（ビューオンリー状態）
2. いずれかのセルを編集
3. 編集がビューのみに反映され、ファイル保存されないことを確認
4. 「📄 Restore Original」で元に戻ることを確認

## 期待される動作

- ✅ **ソートはデフォルトでビューオンリー**（ファイル変更なし）
- ✅ **視覚的インジケーター**でソート状態が分かる
- ✅ **簡単な復元**で元の順序に戻せる
- ✅ **明示的保存**でファイルに反映できる
- ✅ **数値ソート**も正常に動作
- ✅ **編集操作**との整合性が保たれる
