
# キーボードショートカット一覧

## 通常モード（セル編集モード外）

| ショートカット | 動作 |
| --- | --- |
| Enter | 選択セルの編集開始 |
| F2 | 選択セルの編集開始 |
| Tab / Shift+Tab | 次/前のセルに移動 |
| 矢印キー | 上下左右にセル移動 |
| Ctrl+矢印 / Cmd+矢印 | 空白または連続データの端までジャンプ（Excel風） |
| Ctrl+A / Cmd+A | 全セル選択 |
| Ctrl+C / Cmd+C | 選択セルをコピー |
| Ctrl+V / Cmd+V | 貼り付け |
| Ctrl+X / Cmd+X | 切り取り |
| Delete / Backspace | 選択セルの内容をクリア |

## セル編集モード中

| ショートカット | 動作 |
| --- | --- |
| Enter | 編集確定＆下のセルへ（IME確定直後は無効化） |
| Shift+Enter | 改行（テキストエリア時、編集継続） |
| Ctrl+Enter / Cmd+Enter | 編集確定＆編集終了 |
| Tab / Shift+Tab | 編集確定＆次/前のセルへ |
| Escape | 編集確定＆編集終了 |

## ヘッダー編集時

| ショートカット | 動作 |
| --- | --- |
| Enter | 編集確定＆編集終了 |
| Tab / Shift+Tab | 編集確定＆次/前のヘッダーへ |
| Escape | 編集キャンセル |

---
* IME（日本語入力）確定時のEnterはセル確定とならず、入力が継続されます。
* Macの場合はCtrl→Cmd、Backspace→Deleteに読み替えてください。
